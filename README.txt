Untuk mejalankan Web ini ikuti langkah" berikut
=================================================================================================================
1. Disarankan agar terhubung ke internet, dikarenakan beberapa data wajib menggunakan jaringan internet
2. install aplikasi xampp. sesuaikan dengan versi windows apakah 64 bit atau 32 bit
3. setelah selesai, masuk ke direktori tempat penginstallan
4. pilih folder xampp -> htdocs
5. extrak file didalam folder tersebut. perhatikan hasil extrak jangan sampai ada folder yg sama didalam
6. jalankan aplikasi xampp, run apache dan mysql
7. masuk ke browser -> ketikan : localhost/phpmyadmin
8. buat database baru dengan nama webgis
9. pilih database yang baru dibuat tadi
10. klik import -> pilih database yang telah tersedia -> klik go
11. setelah selesai jalankan web dengan ketikkan localhost/webgis_prototype
12. jika berhasil akan muncl halaman depan
13. selamat mencoba
